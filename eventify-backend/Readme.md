A Node.js RESTful API for managing an event management platform with multi-role access (Super Admin, Organizers, and Attendees). Built with scalability and security in mind to handle event creation, ticketing, scheduling, promotions, and attendee engagement.

## ğŸš€ Features

## Super Admin Module

- User & Organizer Management (approve new organizers, suspend accounts, view user/organizer dashboards)
- Event Catalog Management (centralized event database, standardize event categories and types)
- Booking & Transaction Oversight (view all event and ticket bookings, track payment status, generate financial reports)
- Analytics & Reporting (platform usage analytics, organizer performance metrics, revenue reporting)
- Promotions & Discounts (create system-wide promotions, approve/reject organizer-initiated discounts)
- Dispute & Complaint Management (handle user complaints, track resolution history, mediate conflicts)

## Event Organizer Module

- Event Management (create, update, and delete events; manage event details like date, venue, services)
- Booking Management (view and manage event bookings, update booking status, handle cancellations/refunds)
- Promotions & Discounts (set discounts for specific events or ticket types, track promotion performance)
- Reporting & Analytics (sales reports for events, view attendance metrics, analyze revenue and profit margins)
- Customer Engagement (communicate with attendees, respond to inquiries, provide event updates)

## User/Attendee Module

- Account Management (registration, login, profile editing, view booking history)
- Event Discovery (search events by title, category, date, or location; apply filters; view detailed event information)
- Event & Ticket Booking (add events to cart, proceed to checkout, integrated payment gateway, booking confirmation)
- Reviews & Ratings (rate events and organizers, write reviews and feedback)
- Request Unavailable Event Types (notify organizers about desired event types or services not currently offered)
